研究生2.0 02

T-test：比較兩組的平均值

核心問題：這兩組的平均值，有沒有差異？

使用條件：有兩組，而且只有兩組。組別是類別變數——性別、實驗組 vs 控制組、有無接受訓練。

⚠️ 超過兩組就不能用 T-test → 換 ANOVA。硬拆三組跑三次 T-test，整體錯誤率會超標。
⚠️ 前後測是同一批人測兩次 → 用 Paired-sample T-test，不是 Two-sample T-test。
研究生2.0 03

ANOVA：三組以上的平均值比較

One-way ANOVA

一個類別自變項，三組以上。
例如：A、B、C 三種教學法對成績的影響。

⚠️ ANOVA 只說「有差異」，不說哪兩組不同 → 需要做 pairwise comparison + p 值校正。

Two-way ANOVA

同時看兩個類別變數。
例如：性別 × 教學法對結果的影響，還能看交互作用。

研究生2.0 04

Regression：多個變項如何影響結果

核心問題：這些變項，各自對結果有多大的獨立影響力？

同時放入多個變項，在控制其他變項的條件下，估計每一個變項的影響。

例如：「在控制了家庭背景之後，教學法對成績的影響還顯著嗎？」

💡 類別變項轉成 dummy variable 一樣可以丟進 Regression。不是只能用連續變項。
研究生2.0 05

ANOVA 跟 Regression 差在哪？

底層是同一套數學。

ANOVA 的自變項是類別變數；
Regression 通常是連續變數（或 dummy variable）。

用 dummy variable 跑 ANOVA 和 Regression → 結果一樣。
懂了這個，初、中級統計就沒什麼問題了。

研究生2.0 06

卡方檢定：比的是比例，不是平均值

變項是名目變數（nominal），想知道各組的比例是否相同。

例如：男女在「是否吃素」這件事上，比例有沒有差異？ → 卡方。✅

⚠️ 卡方的變項不能是連續變數或序位變數，必須是「是/否」、「男/女」這種名目類別。

🧭 懶人決策流程

1
你想比較的是「平均值」嗎？
↳ 兩組 → T-test
↳ 三組以上 → ANOVA
↳ 前後測 / 配對樣本 → Paired T-test
2
你想知道多個變項如何影響結果？
→ Regression（類別變項先轉 dummy）
3
你比的是「比例」，且變項是名目變數？
→ 卡方檢定
